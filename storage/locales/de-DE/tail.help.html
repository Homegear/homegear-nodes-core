Gibt die letzte Zeile einer Datei aus

Pfad zu einem Verzeichnis/einer Datei oder mehreren Verzeichnissen/Dateien. Werden mehrere Verzeichnisse/Dateien angegeben, so sind diese mit , zu trennen.

Beispiel: /home/user/document.txt, /var/document.txt

Output

file

Gibt den Pfad zur geschriebenen Datei an

payload

Beinhaltet die letzte Zeile der geschriebenen Datei
