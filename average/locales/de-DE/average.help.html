Berechnet den Durchschnitt

Inputs

Nimmt einen Zahlenwert entgegen

Outputs

Gibt den Durchschnitt zurück

Details

Interval

Sammelt alle Eingaben in dem spezifizierten Intervall und bildet den Durchschnitt darüber. Löscht alle Werte, nachdem der Durschnitt gebildet wurde.

Aktuelle Werte

Sammelt von jeder Eingangs Node einen Wert und bildet sofort den Durchschnitt. Löscht Werte einer Node, nachdem keine neuen Werte dieser Node registriert wurden.

Runden des Ergebnisses

Rundet das Ergebnis entsprechend der Einstellung.
Bei Ganzzahl wird immer auf eine ganze Zahl gerundet.
Bei Fließkomma wird das genaue Ergebnis ausgegeben
